# 🎵 Music Festival Website

A modern and responsive Music Festival landing page built using **HTML** and **Tailwind CSS**.

---

## 📌 About The Project

This project is a responsive one-page Music Festival website designed to showcase:

- Event details  
- Artist lineup  
- Ticket pricing  
- Call-to-action sections  

The website is styled entirely using **Tailwind CSS utility classes**, focusing on clean UI design and responsive layout.

---

## 🚀 Features

✅ Fully responsive design  
✅ Modern UI using Tailwind CSS  
✅ Festival lineup section  
✅ Ticket pricing cards  
✅ Navigation bar  
✅ Call-to-action section  
✅ Clean and minimal layout
